# Python Self-Learning: From Zero to AI Engineer

This repository is a structured, 43-day curriculum designed for a highly
motivated beginner, Aditya, to go from zero Python knowledge to a level
of proficiency required for serious work in AGI, robotics, and space engineering.

## The Philosophy

This is a master class, not a cheatsheet. The goal is deep, foundational
understanding. We learn *why* things work, what breaks them, and how the
concepts connect to real-world, high-stakes engineering. The curriculum is
learned by doing, with a heavy emphasis on hands-on coding, problem-solving,
and building.

## The 4-File System

Each day is a self-contained module with four distinct files:

1. `0_ignite.py` — The daily launch sequence. Recap of yesterday, warm-up
   problems to prove you remember, and a bridge to today's topic.
2. `1_nexus.py` — The master class lecture. Exhaustive, in-depth explanation
   broken into 8 layers: The Problem, The Concept, The Mechanics, The Wrong Way,
   Real Examples, Pause and Think, Edge Cases, and The Connection.
3. `2_gauntlet.py` — The training ground. 10+ problems in 3 tiers:
   Warm Up, Level Up, Beast Mode. No solutions provided.
4. `3_forge.py` — The workshop. Real projects to build using today's concepts,
   with hints, assert tests, extensions, and professional context.

Your work lives in `My_Work/` which mirrors `AI_Material/` exactly.

## Daily Habit (Non-Negotiable)

After your daily session — 30 to 45 minutes on HackerRank Python track.
Easy problems only until Day 20. This is your practice system running
parallel to the course.
